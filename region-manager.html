Region, City and Township Manager

Magento 2

Feature

  1. Able to add/edit Region, City, Township in admin panel.
  2. Able to change address format by update customer address template.
  3. Can show/hide township in case you don't want to use township.
  4. Can hide country field from checkout process.
  5. Easy to translate by change region, city, township label in admin for multi-stores.
  6. Can import region, city, township from csv.
  7. Make city field show as dropdown so customer easier to choose.
  8. City data will show depend on the region, township data will show depend on city data.
  9. The city select options will change right after you select region, no need to wait ajax load data becasue I dont use ajax.
  10. Optimized code to make sure that website speed always fast (tested with more than 1600 cities and 42000 townships).
